To run this project read the following instructions

1. Run using cmd
	- open the project directory
	- navigate to src folder
	- open cmd in the src folder
	- type command 'python Finalcode.py'

2. Run using VS Code
	- open the project directory
	- run Finalcode.py file in terminal

after executing Finalcode.py file you will get three options
1] dehaze a single image
2] dehaze a video
3] dehaze all images in a directory
choose option accordingly
